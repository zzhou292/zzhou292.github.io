Drake

Model-Based Design and Verification for Robotics.

Accomplishments
  • Tools: C++, CUDA, DDS
  • C++ toolbox started by the Robot Locomotion Group at MIT CSAIL. Core development led by the Toyota Research Institute
  • Simulates complex robot dynamics (friction, contact, aerodynamics). Exposes structure in governing equations (sparsity, analytical gradients, etc.). Provides advanced planning, control, and analysis algorithms
  • Python interface for rapid prototyping. Solid open-source implementations of state-of-the-art algorithms

Robot Wheel-Terrain Interaction Simulation For Control Design

Terra-Mechanics Soil Modeling Pipeline for Wheel-Terrain Interaction in Off-Road Robotics Applications

Accomplishments
  • Tools: C++, C, DDS, MPI, CUDA, OpenCL, HPC
  • Modeling soil behavior, especially in the application of robotics wheel-terrain interaction simulation for the digital solution of traction and mobility control validation.
  • Development of a semi-empirical Soil Contact Model based on Bekker-Wong theory, as well as a continuum representation utilizing Smooth Particle Hydrodynamics (SPH) and the Discrete Element Method (DEM) for simulating granular terrain.
